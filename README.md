# 📈 퀀트 라이브 스크리너 (Quant Live Screener)

한국(KOSPI·KOSDAQ) + 미국(NYSE·NASDAQ) 주식을 **4대 학술 팩터**로 점수화하고,
**근실시간 시세**로 순위를 갱신하는 Streamlit 대시보드.

> 30년차 엔지니어의 견고한 아키텍처 + 행동경제학의 통찰을 한 화면에.
> 각 팩터가 *왜* 초과수익을 내는지를 **투자자 심리(편향)** 로 해설합니다.

---

## 핵심 기능

| 영역 | 내용 |
|------|------|
| **시장** | 한국(`FinanceDataReader`/`pykrx`) · 미국(`yfinance`) — 동일 인터페이스로 추상화 |
| **팩터** | 가치(저PER·저PBR·고배당) · 모멘텀(12-1) · 퀄리티(ROE 등) · 저변동성 |
| **표준화** | 윈저라이즈 → 횡단면 z-score → 가중 종합점수·백분위 랭킹 |
| **전략** | 멀티팩터 / 행동재무 / 가치투자 / 퀄리티 프리셋 + **커스텀 가중치 빌더** |
| **실시간** | `st.fragment(run_every=N)` 로 무거운 일봉은 캐시, 시세만 N초마다 폴링 |
| **행동재무** | 팩터별 편향 해설 + 종목별 위험 플래그(🔥과열·🎰복권형·⚠️가치함정) |

## 빠른 시작 (Windows)

```powershell
# 1) 의존성 설치 (최초 1회)
python -m pip install -r requirements.txt

# 2) 실행 — 둘 중 하나
.\run.bat
#   또는
python -m streamlit run app.py
```

브라우저에서 `http://localhost:8501` 이 자동으로 열립니다.
좌측 사이드바에서 **시장 · 전략 · 필터 · 폴링 주기**를 조절하세요.

> 최초 적재는 종목 일봉·펀더멘털을 받아오느라 다소 걸립니다(특히 한국 시장의
> 종목별 일봉 루프). 이후 6시간 동안 캐시되어 즉시 뜹니다.
> 사이드바의 **♻️ 데이터 새로 적재**로 캐시를 비울 수 있습니다.

## 다른 사람과 공유 (외부 접속)

1. **임시 공개 링크 (Cloudflare 터널) — 추천·가장 빠름**
   `share.bat` 실행 → 콘솔에 뜨는 `https://xxxx.trycloudflare.com` 주소를 전달.
   내 PC와 그 창이 켜져 있는 동안만 유효하고, 재시작하면 주소가 바뀝니다.
   > 이 네트워크는 UDP(QUIC/7844)가 막혀 있어 `cloudflared` 가 기본 QUIC로는 연결 실패합니다.
   > 그래서 `share.bat` 은 `--protocol http2`(TCP)로 연결하도록 박아두었습니다.
2. **같은 와이파이(LAN)** — `streamlit run` 시 출력되는 Network URL
   (예: `http://172.30.1.53:8501`)을 같은 네트워크 사용자에게 공유(방화벽 8501 인바운드 허용 필요).
3. **영구 웹사이트 (Streamlit Community Cloud)** — GitHub 푸시 후 share.streamlit.io에서 배포하면
   `xxx.streamlit.app` 주소가 24시간 유지(내 PC 불필요).
   - 미국 서버라 **KRX·네이버가 차단**되므로, 한국 시장은 자동으로 **yfinance(.KS/.KQ)** 폴백 경로로 받는다
     (`korea.py` 이중 경로 + `kr_universe.py` 정적 목록). 모멘텀·저변동성·ROE는 정상, PER/PBR은 약해질 수 있음.
   - 클라우드(비-Windows)에서는 기본 시장이 자동으로 **미국**으로 잡힌다(한국도 사이드바에서 선택 가능).

## 4대 팩터와 행동경제학적 근거

| 팩터 | 부호 방향 | 대표 편향 | 핵심 레퍼런스 |
|------|-----------|-----------|----------------|
| **가치** | 쌀수록 ↑ | 과잉반응·외삽 편향 | De Bondt & Thaler (1985); LSV (1994) |
| **모멘텀** | 추세 강할수록 ↑ | 과소반응·군집·처분효과 | Jegadeesh & Titman (1993); Hong & Stein (1999) |
| **퀄리티** | 우량할수록 ↑ | 제한된 주의 | Sloan (1996); Novy-Marx (2013) |
| **저변동성** | 변동성 낮을수록 ↑ | 복권선호·과신 | Baker–Bradley–Wurgler (2011) |

효율적 시장이라면 이 초과수익은 0이어야 합니다. 수십 년간 사라지지 않는다는
사실 자체가, 시장가격이 *합리적 기대*가 아니라 *편향된 다수의 심리*로
형성된다는 행동경제학의 핵심 주장을 뒷받침합니다.

## 아키텍처

```
quant-screener/
├── app.py                  # Streamlit UI (사이드바 → 적재 → 점수 → 실시간 폴링)
├── config.py               # 전역 파라미터
├── quant/
│   ├── factors.py          # z-score·윈저라이즈·모멘텀·변동성·팩터 패널
│   ├── strategies.py       # 프리셋 가중치 + 커스텀 빌더
│   ├── behavioral.py       # 팩터별 편향 해설 + 종목 행동 플래그
│   ├── screener.py         # 종합점수·랭킹·필터
│   └── datasource/
│       ├── base.py         # 공통 DataSource 인터페이스(ABC)
│       ├── korea.py        # pykrx + FinanceDataReader 어댑터
│       ├── us.py           # yfinance 어댑터
│       └── universe.py     # 미국 유니버스 폴백 목록
└── tests/test_factors.py   # 오프라인 팩터 수학 검증
```

새 시장(예: 일본·홍콩)을 추가하려면 `DataSource`를 구현하고
`datasource/__init__.py`의 팩토리에 등록하면 됩니다. 나머지 코드는 무수정.

## 데이터·실시간성 관련 주의

- **무료 데이터** 기반이라 시세는 **지연·결측**이 있을 수 있습니다(진정한 틱 단위 아님).
  실시간 틱이 필요하면 `quant/datasource/`에 유료 API(WebSocket) 어댑터를 추가하세요.
- 미국 펀더멘털은 `yfinance`의 종목별 `info` 호출이라 최초 적재가 느릴 수 있습니다(스레드풀 병렬·캐시 적용).
- 한국 부채비율·이익률은 무료 일괄 제공이 어려워 퀄리티를 **ROE(EPS/BPS)** 로 근사합니다.

> ⚠️ 본 프로젝트는 **리서치/교육용 도구**이며 투자 권유가 아닙니다.
> 모든 투자 판단과 책임은 사용자 본인에게 있습니다.
